🏚️
📜 วิธีเล่น
'คุณ' มาสมัครเป็นยามเฝ้าถุงเงินให้บีเซนเซย์

คุณมีหน้าที่แยกถุงเงินจากถุงทองผสมกรวด หากคุณได้ถุงทองผสมกรวดคุณจะต้องผันคำคุณศัพท์เพื่อแยกทองออกจากกรวด และต้องระวังโจร หากถูกโจรขโมยเงินไป เงินก็จะหายไปเรื่อย

เมื่อครบ 3 วัน ใครได้เงินน้อยสุด ต้องถูกลงโทษ!
💰
เลือก 1 กอง ถ้าเจอถุงเงิน → รับ 300,000฿
🪙
เจอทองผสมกรวด → ผันคำคุณศัพท์
ตอบถูก = +200,000฿ | ตอบผิด = +0฿
🦹
รอบโจร! ตอบผิด → เสียเงิน + หัวใจ
⚔️
ชนะหัวหน้าโจร → ได้อาวุธ ป้องกันการโจมตี
❤️
หัวใจ 5 ดวง หมด = แพ้ทันที
🏆
อยู่รอด 3 คืน + เงินมากสุด = ชนะ!
⚠️ ผู้ที่เงินน้อยสุดเมื่อจบเกม จะต้องผันคำคุณศัพท์ 70 คำ!
